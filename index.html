Learn from Code
 Please try learning from code.

In order to do so just imagine a situation in which you would not have any description (explanation) to code and should learn everything only from the code itself, analyzing the code and the result. Learning programming this way, step by step, is called learning from code.

This approach may be very effective.

So you can practice the most extreme approach here: If some code is clear to you, ignore all of my explanations to it.

Try learning from code as much as possible.
